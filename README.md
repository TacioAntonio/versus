# <img src="./images/versus.png" width="62" height="30"> Versus

![image](images/multiplayer-gaming-platform.gif)
> A platform for developing multiplayer games

## Installation
### Front-end
```sh
$ git clone https://github.com/TacioAntonio/versus
$ cd versus/frontend
$ npm i
```

### Back-end
```sh
$ git clone https://github.com/TacioAntonio/versus
$ cd versus/backend
$ npm i
```

## Running locally
```sh
$ npm start
```
